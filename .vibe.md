Где автобус? — Android-виджет и приложение для отслеживания прибытия городских автобусов в реальном времени по остановкам Москвы.

Стек: Kotlin (Android, minSdk 24), FastAPI (Python) на бэкенде, деплой в Yandex Cloud Serverless Containers. Данные берутся парсингом SSR-состояния из HTML Яндекс.Карт — официального API нет.

Сейчас в фокусе: home screen виджет с живым обратным отсчётом ETA, фильтрация по маршрутам, выбор остановки через WebView с Яндекс.Картами. Недавно: починили виджет (RemoteViews-ограничения), добавили кастомную иконку, упростили добавление остановок до единственного способа — через карту.

Архитектурная специфика: бэкенд stateless, список остановок в env-переменной JSON-массивом без базы данных. Виджет опрашивает сервер каждые 15 секунд, но не сбрасывает обратный отсчёт при одинаковом ответе — клиент интерполирует ETA между запросами.
